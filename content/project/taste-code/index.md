## Overview

TasteCode is a modern Android recipe app built with Kotlin and Jetpack Compose — designed for fast recipe search that still works when you're offline.

## Features

- Real-time recipe search
- Offline storage via **Room Database**
- Filtering by ingredient, cuisine, and more
- User authentication workflows powered by **Firebase Firestore**

## Stack

Kotlin, Jetpack Compose, Firebase Firestore, Room Database, Android Studio.
